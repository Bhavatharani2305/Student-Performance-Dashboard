# Student-Performance-Dashboard
Student Performance Dashboard built using Power BI to analyze marks, attendance, subject-wise performance, and student rankings with interactive visualizations and filters.


 📌 Overview

This project is an interactive **Student Performance Dashboard** developed using Microsoft Power BI. It helps analyze student academic data to identify performance trends, subject difficulty levels, and the impact of attendance on marks.

---

## 📊 Dataset Information

The dataset contains **100 student records** with the following fields:

* Student Name
* Class
* Subject
* Marks
* Attendance
* Gender
* Exam Date
* Teacher

---

## 📈 Dashboard Features

### 🔹 KPI Cards

* Total Number of Students
* Average Marks
* Average Attendance

### 🔹 Visualizations

* 📊 Average Marks by Subject (Bar Chart)
* 📊 Marks Distribution (Column Chart / Histogram)
* 📊 Student Ranking by Marks (Table / Bar Chart)
* 📊 Marks by Gender (Pie Chart)
* 📊 Subject-wise Performance (Column Chart)

### 🔹 Scatter Analysis

* Attendance vs Marks Scatter Plot
* Shows relationship between attendance and academic performance

### 🔹 Interactive Slicers

* Class
* Subject
* Gender
* Teacher

These filters allow dynamic and real-time data analysis.

---

## 📌 Key Insights

* Students with higher attendance tend to score better marks
* Certain subjects have lower average scores, indicating higher difficulty
* Performance varies across classes and teachers
* Gender-based performance is relatively balanced
